# Interior Design Website

A modern and responsive interior design website built using **React.js**, **TailwindCSS**, and **Framer Motion**.  
This website showcases different design services, artwork collections, and provides a contact form with map integration.  

---

## 🚀 Features
- **What We Do** section with animated service cards  
- **Get a Quick Estimate** form for user inquiries  
- **Get in Touch** section with embedded Google Maps  
- **Footer** with Contact Us and social media icons (Instagram, Facebook)  
- Fully **responsive** design for mobile, tablet, and desktop  

---

## 🛠️ Tech Stack
- [React.js](https://reactjs.org/)  
- [TailwindCSS](https://tailwindcss.com/)  
- [Framer Motion](https://www.framer.com/motion/)  
- [React Icons](https://react-icons.github.io/react-icons/)
